--- title: Minimum Eccentricity Shortest Paths in some Structured Graph Classes authors: - F. Dragan - A. Leitert key: 2016-04-Dra-Lei type: J journal: Journal of Graph Algorithms and Applications volume: 20 (2) pages: 299-322 year: 2016 doi: 10.7155/jgaa.00394 arXiv: 1511.05109 --- We investigate the Minimum Eccentricity Shortest Path problem in some structured graph classes. It asks for a given graph to find a shortest path with minimum eccentricity. Although it is NP-hard in general graphs, we demonstrate that a minimum eccentricity shortest path can be found in linear time for distance-hereditary graphs (generalizing the previous result for trees) and in $\mathcal{O}(n^3m)$ time for chordal graphs and for dually chordal graphs.
